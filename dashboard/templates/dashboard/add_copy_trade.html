How It Works

Trader-Centric: Add ONE trade per trader (not per user)

Auto-Distribution: All users copying this trader see the trade

Proportional P/L: Each user's profit/loss is calculated based on their investment

Market: Select the trading pair or asset

Direction: Buy (Long) or Sell (Short)

Base Amount: The trader's investment amount (for reference)

P/L Percentage: Enter as percentage (e.g., 15.50 for +15.5%, -8.25 for -8.25%)

Status: Open (active) or Closed (completed with balance update)
